Sistem ini dirancang untuk membantu admin dalam mencatat data peminjaman buku di perpustakaan. Data peminjaman akan diinput setelah mahasiswa melakukan peminjaman dan pengembalian buku, sehingga seluruh riwayat peminjaman dapat tersimpan dengan baik.

#FITUR SISTEM
1. login admin
2. input data peminjaman
3. edit dan hapus data peminjaman
4. melihat data peminjaman

#TEKNOLOGI YANG DIGUNAKAN
1. PHP
2. HTML,CSS
3. MYSQL
4. XAMPP

#Cara Menjalankan Sistem
Langkah-langkah menjalankan sistem:
1. Install dan jalankan XAMPP
    Aktifkan:
    *Apache
    *MySQL

2. Simpan project
   Letakkan folder project di:
    C:\xampp\htdocs\perpustakaan

3. Import database
   Buka http://localhost/phpmyadmin
   Buat database
   Jalankan query (tabel users, buku, peminjaman)

4. Jalankan di browser
   http://localhost/perpustakaan/login.php

5. Login sebagai admin
   Username: admin
   Password: 123
